# PersonalFinanceDashboard

This project is a Personal Finance Dashboard builT using Excel and SQL to help users track, analyse, and improve their financial decisions.

It allows users to record daily transactions, generate weekly/monthly reports, track budgets, and gain insights into spending patterns.

## Problem Statement

Managing personal finances manually is difficult. Most people:
- Don't track daily expenses
- Don't know where their money goes
- Struggle to save consistently

## Solution

This system provides:
- A structured way to store financial data (SQL)
- A dynamic dashboard to analyse and visualise data(Excel)

## Feature
## 📑 Data Management
- Add daily income and expense records
- Store data in SQL database.
  
## 📊 Reporting & Analysis
- Daily, Weekly and Monthly spending reports
- Income vs Expense comparison
- Category-wise expense breakdown.
- Investment and Saving calculation and tracking

## 🎯 Budget Tracking
- Set monthly budget
- Compare actual spending vs budget
- Identify overspending

## 🧠 KPI
- Current Balance
- Monthly Income
- Monthly Expense
- Monthly Saving / Investment

## Key Learnings
- Integrated SQL with Excel for real-world data workflows.
- Build dynamic dashboards using Power Query.
- Applied data analysis techniques to derive insights
- Improved problem-solving and data modelling skills.
