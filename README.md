# 💰 Personal Finance Dashboard

A **data-driven Personal Finance Dashboard** built using **Excel and SQL** to help individuals track, analyse, and improve their financial decisions.

This project enables users to record daily transactions, generate periodic reports, monitor budgets, and gain actionable insights into their spending behaviour.



## 🚩 Problem Statement

Managing personal finances effectively is difficult without a clear and structured system. Most individuals struggle to track where their money goes, identify spending patterns, and maintain a healthy financial balance.

In the absence of proper tracking and analysis:

* Expenses go unnoticed
* Spending habits remain unclear
* Budget control becomes ineffective
* Financial decisions are often made without data

This project addresses these challenges by building a **Personal Finance Dashboard using Excel and SQL**, which transforms raw transaction data into meaningful insights.

The system focuses on analysing key financial indicators such as:

* **Monthly Income** – Total earnings per month
* **Monthly Expenses** – Total spending across categories
* **Current Balance** – Net remaining balance
* **Spending Trends** – Daily, weekly, and monthly patterns
* **Category-wise Expenses** – Detailed breakdown of spending

---

## ✅ Solution

This project provides an end-to-end solution for personal finance management:

* **SQL** is used to store, clean, and analyse transaction data
* **Excel** is used to create an interactive and dynamic dashboard
* Automated queries generate insights for better decision-making
